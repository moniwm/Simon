For the creation of the program, the first modules had to be identified with their functionalities.
It is needed to build the interfaces that will draw, update and handle the events on the screen
Then it will be necessary to build the base of the levels, it would be added randomly in a linked list,
then the event will be detected when the user presses a color, each color will be assigned to a
string to be compared sequentially with the linked list created. In addition, each color pressed will be
added to the linked list, that way when the element of position "i" of both lists is compared and they are
different, the game will be over.
